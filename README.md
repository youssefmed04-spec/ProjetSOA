### ***Description du projet :*** 

c'est une interface de gestion de personnes qui contient :

* Une fonction qui affiche la liste des personnes
* Une fonction qui ajoute une personne 
* Une fonction qui fait la recherche d'une personne a travers un ID donné 
* Une fonction qui modifie une personne 
* Une fonction qui supprime une personne 

### ***Technologies utilisées***

Cette interface est développée par HTML\\JS\\JSP (le fichier index.jsp).

Elle est liée par un backend développé par : REST en utilisant JAX-RS avec JPA(Hibernate),Ce backend expose des services REST permettant la réalisation des différentes fonctionnalités affichées sur l'interface 

### ***Instructions pour exécuter le projet :***

Comme indiqué dans le vidéo j'ai lancé le projet à travers Eclipse IDE. Donc pour lancer le projet :

Clique droite sur le dossier du projet

Vous voyez par suite l'interface est ouverte
